My response to this case study was prepared with the following assumptions in mind.
    • Short timeframe. The summative math assessment is coming up quickly. The principal needs to do something even more quickly in order to have a chance of improving performance. I've priorizied actionability and fast turnaround over exploring every possible angle.
    • Comparable historical data. I've been informed that, for this case study, I should assume robust data regarding the current 5th grade class. I am also assuming equally robust and comparable data regarding previous 5th grade classes - some degree of year-over-year consistency in when and how indicators are measured.
    • Bottom line over methodology. I'm assuming the principal doesn't really care about how I reached my conclusions as long as I can succinctly explain why I was justified in reaching them. I've left out almost all details about the methods themselves, focusing instead on the results.
    • In-person contact. This site is barely a stand-alone document. I designed it that way. In a real-world scenario I would use this site as a read-ahead and as a refernce to facilitate an in-person conversation with the principal.
    • Meeting in the middle. While I never assume that anyone wants jargon, I do assume the principal is willing to become an educated consumer of analysis. I can and will explain any concept or procedure a person wants to understand, but I've seen little good come from analogizing a technical concept to the point that a consumer thinks he or she does not need to ask questions about it.

Introduction [top]

This analysis indexes scholars' potential for improvement. Focus your efforts first on those scholars with the lowest index value. Those scholars are mostly likely to respond with higher performance. Don't devote extra attention to scholars with high index values unless lower-indexed scholars' needs have already been addressed.


Problem

  Scholars in Middle School 1 under-performing across most academic and cultural indicators

  Little time to implement interventions/changes (major summative math assessment approaching)

Resources

  ~250 potential predictors of summative assessment performance (tests, quizes, class work, attendance, homework, paycheck balance, etc.)

  Indicators and summative assessment performance data available across ~800 scholars per year for years 2010 through 2013

  Same indicators and performance data available across ~600 current scholars, including 79 scholars at Middle School 1

Process

  Filter indictors to only include those that have been consistently predictive of summative assessment performance

  Use filtered indicators to measure similarities among all current 5th grade scholar (across all middle schools)

  Strain out noise in similarity data to get a few basic dimensions of scholar performance

  Cluster scholars based on where they map onto the basic dimensions

  Identify misclassificatons - scholars who group with high performers but who are themselves under-performing - those scholars are the best candidates for additional attention/help

Filter Indicators [top]

  Organize historical scholar data into a table with scholar/year as rows and indicators as columns

  Map each table record to eventual score in summative assessment

  Build statistical model to score each indicator in terms of its informativeness in predicting summative assessment performance

  Unsurprisingly, math-related predictors tend to be most informative, but several others have enough value to retain them

  Methods note: training the model on all students over multiple years limits the chance of idiosyncracies being mistaken for reliable predictors


The following graph shows a high-level view of which indicators were most useful in predicting performance on the summative assessment. Each indicator has multiple values because indicators are repeated over the school year (multiple quizzes, multiple weeks of attendance measures, etc.) The colors show which indicators were in the top percentile of predictive power, which were in the next decile, and then the next two deciles, and then all the rest. You can see that some indicators had a single most important exemplar while other indicators were predictive when taken all together. Also note that while most informative indicators came, unsurprisingly, from the math category, some other academic and cultural variables were important as well.

Map Similarities [top]

  Organize current scholar data into a table with retained indicators as rows and individual scholars as columns

  Calculate similarity (correlation) between every pair of scholars

  Statistically create indices that account for the greatest amont of differences within the similarity network

  Cluster scholars into groups that reflect major differences in groups of attributes

  Identify clusters that are majority high-performing but contain low performers from Middle School 1

This graph visualizes scholar similarities as a network cloud. It puckers in many different spots - those are individual scholars. Notice the cloud looks more dense in some places: a particularly dark area runs from the bottom-left to the top-right. That indicates that there is some structure to student similarities that can be mined for insights.


We flaten the network into a matrix of similarity estimates that can be further analyzed. The darker the orange area, the stronger the positive correlation between students; the darker the grey area, the stronger the negative correlation. Students in this graph have been ordered by the most important component of the noise reduction procedures we used to filter the data - the order shows some initial clustering of scholars.


The first two components from the noise-reduction procedure capture over 92% of all the information from the network cloud. Using these first two components to map scholar similarity enables the use of a cluster analysis, which produces 15 clusters, in each of which scholars are more similar to each other than they are to scholars in other clusters.

Prioritize Scholars [top]

  Calculate difference between current scholar performance and cluster average

  Resulting index prioritizes scholars in terms of which students are most likely to improve the greatest amount in response to extra help and attention

  The more negative an index value, the more that student is under-performing relative to cluster average

Other Options [top]

  A more involved analysis would create model predictions for each scholar instead of using cluster averages. This might priortize scholars differently, and would include probability estimates for each prediction, but would require more time (3-4 weeks to get results).

  Intial model results and clusters from current analysis can be mined to suggest drivers of student performance in each cluster. However, it may be more effecient to just ask targeted scholars directly to determine reasons for underperformance.

  It's important to get feedback on how efforts based on this analysis turn out. If we can identify cases in which this analysis fails, we can develop more useful indicators for future use.
